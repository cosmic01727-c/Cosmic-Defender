# Cosmic Defender
Cosmic Defender is a game where you shoot aliens.
## Emulators:
[Javatari](https://javatari.org)

[Stella](https://stella-emu.github.io)

[RetroArch](https://www.retroarch.com)

## Compiling:
```bash
dasm cosmic_defender.asm -f3 -v0 -ocosmic_defender.bin
```

## Running the game
Install one emulator at the top you like and run the bin file.
